# 东京观景台一日游：轻松俯瞰城市的浪漫

想要用最简单的方式俯瞰东京？其实不必赶早起爬山，市中心就有许多观景台，可以在轻松的下午来一场高空散步。以下推荐几个适合“懒人”的观景台路线。

---

## 1. 东京晴空塔（Tokyo Skytree）
- **高度**：634米，日本第一高塔  
- **特色**：晴天能远眺富士山，夜晚能看到整片东京夜景  
- **贴士**：可以买联票顺便去水族馆，避免只排队看景  

📍 [Google Maps - 东京晴空塔](https://www.google.com/maps?q=Tokyo+Skytree)

---

## 2. 六本木之丘・森大厦观景台
- **高度**：238米  
- **特色**：无遮挡的天台，夜景极美；旁边还有森美术馆  
- **贴士**：建议傍晚去，能同时看到夕阳和夜景的切换  

📍 [Google Maps - 六本木之丘](https://www.google.com/maps?q=Roppongi+Hills)

---

## 3. 东京都厅舍（新宿）
- **高度**：202米  
- **特色**：**完全免费！** 北展望室还能看到富士山  
- **贴士**：比起付费景点，这里性价比最高，游客不多  

📍 [Google Maps - 东京都厅舍](https://www.google.com/maps?q=Tokyo+Metropolitan+Government+Building)

---

## 4. 涩谷Sky
- **高度**：229米  
- **特色**：位于涩谷站上方，整个天台开放式玻璃围栏，和朋友拍照非常出片  
- **贴士**：强烈推荐黄昏时分去，天空颜色特别美  

📍 [Google Maps - 涩谷Sky](https://www.google.com/maps?q=Shibuya+Sky)

---

## 小结
如果你时间紧凑，推荐优先顺序是：  
1. **涩谷Sky**（氛围最好，照片好看）  
2. **东京都厅舍**（免费又方便）  
3. **晴空塔**（经典，但人多票贵）
