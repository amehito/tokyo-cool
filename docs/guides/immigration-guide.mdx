# 🛬 日本入境流程攻略

第一次来日本？别担心，这里有详细的入境流程 + 常见问答句子 + Visit Japan Web 模版，帮你轻松应对。

---

## 📑 入境审查（Immigration）

### 需要准备
- 护照  
- 签证 / 在留卡（如有）  
- **入境卡（已电子化）** 👉 建议提前在 [Visit Japan Web](https://vjw-lp.digital.go.jp/) 填写  

---

### Visit Japan Web 模版（示例填写）

- **滞在目的**：観光（观光） / 商用（商务）  
- **滞在先住所**：ホテル名 + 地址（可提前准备好）  
- **滞在期间**：2025年8月18日～2025年8月25日（例）  

💡 **Tips**：系统会生成一个二维码，到机场直接出示给工作人员即可。

---

### 常见问答（日语 + 罗马音 + 中文）

1. **来日の目的は何ですか？**  
   Rai-nichi no mokuteki wa nan desu ka?  
   👉 来日本的目的是什么？  
   ✅ 回答：観光です（Kankō desu）= 观光  

2. **どのくらい滞在しますか？**  
   Dono kurai taizai shimasu ka?  
   👉 打算停留多久？  
   ✅ 回答：5日間です（Itsuka-kan desu）= 5天  

3. **どこに泊まりますか？**  
   Doko ni tomarimasu ka?  
   👉 住在哪里？  
   ✅ 回答：◯◯ホテルです（◯◯ hoteru desu）= 在◯◯酒店  

4. **職業は何ですか？**  
   Shokugyō wa nan desu ka?  
   👉 职业是什么？  
   ✅ 回答：エンジニアです（Enjinia desu）= 工程师  

5. **現金はいくら持っていますか？**  
   Genkin wa ikura motte imasu ka?  
   👉 携带多少现金？  
   ✅ 回答：10万円です（Jū-man en desu）= 10万日元

## 🎒 行李提取（Baggage Claim）

- 根据航班号找到传送带。  
- **日语：荷物はどこですか？**  
  Nimotsu wa doko desu ka?  
  👉 行李在哪里？  

---

## 🛂 海关申报（Customs）

- 已在 Visit Japan Web 填写可走「蓝色通道」。  
- 携带超过 100万日元 / 大量烟酒食品需要申报。  

**日语常用句：**  
- 申告するものはありません。  
  Shinkoku suru mono wa arimasen.  
  👉 没有需要申报的物品。  

---

## 🚉 出口与交通（Transport）

- 🚄 成田机场：京成 Skyliner / JR 成田特快  
- 🚄 羽田机场：京急线 / 东京单轨  
- 🚌 机场大巴（リムジンバス）  
- 🚕 出租车  

**日语常用句：**  
- 東京駅までお願いします。  
  Tōkyō-eki made onegaishimasu.  
  👉 请到东京站。  

---

## 💡 小贴士

- 提前在 Visit Japan Web 完成入境卡 + 海关申报，可扫码快速通关。  
- 入境官问问题时不用紧张，**直接用上面这些日语短句回答即可**。  
- 建议准备酒店预订单截图，遇到问题时可直接出示。  
- 入境后第一件事：买一张 **交通 IC 卡（Suica / PASMO）**。
